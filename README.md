# Üç ve Dörde Tam Bölünen Sayıların Ortalamasını Hesaplayan Program

Girilen sayıdan 0'a kadar olan sayılardan hem 3'e hem 4'e tam bölünen sayıların ortalamasını hesaplayan Java programı.

## Proje Yapısı

src/main/java dizini altında yer almaktadır.

## Nasıl Çalıştırılır

src/main/java klasöründeki Main.java dosyasını derleyip çalıştırınız.
